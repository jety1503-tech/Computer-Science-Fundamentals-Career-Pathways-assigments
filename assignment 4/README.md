# Static Portfolio Webpage

## Description
A static personal portfolio webpage built with HTML and CSS. It demonstrates basic front-end layout and a simple Git/GitHub workflow.

## Installation
1. Clone the repository:
git clone https://github.com/jety1503-tech/Computer-Science-Fundamentals-Career-Pathways-assigments/tree/main/assignment%204
2. Open the folder in VS Code.
3. Open `index.html` in your browser.

## Usage
- Edit `index.html` and `styles.css` to customize your name, content, colors, and projects.
- Refresh the browser to see your changes.

## Features
- Responsive layout with simple navigation.
- Sections for About, Projects, Skills, and Contact.
- Clean, modern styling using only HTML and CSS.
- No external frameworks required.
